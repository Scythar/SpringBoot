✅How does Redis work?

ANSWER:
Redis is not just key-value—it supports: String Hash List Set Sorted Set

Redis uses single thread for commands
Avoids:
Locks
Race conditions

👉 Still handles thousands of requests/sec

Where Redis is Used
Caching (most common)
Session storage
Rate limiting
Leaderboards
Messaging
